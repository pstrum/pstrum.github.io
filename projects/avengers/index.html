Avengers Prototype Project

I created a home page based on a given design comprehensive with my friend Hollis in 3 days. What I really enjoy highlighting is the CSS. The style sheet is only 1250 lines of code and features excellent use of Flexbox, @media rules, and great examples of the block model and positioning. Here are a few components that are awesome (with links to GitHub):

  • Portfolio Slider. Notice the use of absolute positioning, aria, and responsive considerations.
  • Portfolio Items. Notice the use of :hover and the :after pseudo class to limit extraneous images from the HTML.
  • Services Components
  • Pricing Components
